# Digitize Ideas - Static Website

## Overview

**Digitize Ideas** is a static website for a digital transformation company that specializes in turning innovative ideas into reality through cutting-edge digital solutions. The website showcases the company's services, projects, team, and mission to help clients bring their concepts to life.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Multi-page Structure**: Organized navigation with dedicated pages for Home, About, Projects, and Services
- **Interactive Elements**: Includes forms, embedded videos, and hover effects for enhanced user engagement
- **Modern UI/UX**: Clean, professional design with a green color scheme and intuitive navigation
- **Team Showcase**: Dedicated section highlighting the company's team members and their roles
- **Project Portfolio**: Grid-based display of completed projects with descriptions and images
- **Service Offerings**: Comprehensive list of digital services including web development, UI/UX design, mobile apps, and more

## Project Structure

```
web/
├── Home.html          # Main landing page with hero section and contact form
├── about.html         # Company story, mission, and team information
├── projects.html      # Portfolio of completed digital projects
├── services.html      # Detailed list of offered services
├── styles.css         # Centralized CSS styling for all pages
└── README.md          # Project documentation
```

## Technologies Used

- **HTML5**: Semantic markup for content structure
- **CSS3**: Responsive styling with flexbox, grid, and media queries
- **Remixicon**: Icon library for visual elements
- **Unsplash**: High-quality stock images for project showcases

## Pages Overview

### Home Page (Home.html)
- Hero section with company branding and tagline
- Interactive video play button
- Contact form with validation-ready fields
- Embedded YouTube video
- Navigation to other sections

### About Page (about.html)
- Company story and mission statement
- Team member profiles with photos and roles
- Professional layout with image and text sections

### Projects Page (projects.html)
- Grid layout showcasing completed projects
- Project cards with images, titles, and descriptions
- Hover effects and call-to-action buttons
- Diverse project types (e-commerce, mobile apps, analytics, etc.)

### Services Page (services.html)
- Service offerings in card format
- Icons representing different service categories
- Detailed descriptions of each service
- Hover animations for interactive feel

## How to Run

1. **Clone or Download**: Place all files in a web-accessible directory
2. **Open in Browser**: Open `Home.html` in any modern web browser
3. **Navigate**: Use the navigation menu to explore different sections

No server setup required - this is a pure static website that runs directly in the browser.
